Fix: Bot Deafen Issue
If the bot is staying deafened (no audio output), the fix is in your voice connection code. The selfDeaf option must be false.
// ❌ Wrong - bot can't hear/output audio joinVoiceChannel({ channelId: channel.id, guildId: guild.id, adapterCreator: guild.voiceAdapterCreator, selfDeaf: true, // ← this is the problem selfMute: false, }); // ✅ Correct joinVoiceChannel({ channelId: channel.id, guildId: guild.id, adapterCreator: guild.voiceAdapterCreator, selfDeaf: false, // ← audio flows correctly selfMute: false, });
